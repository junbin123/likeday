## 快速开始
运行此项目，需自行注册微信小程序
```shell
# 下载项目
git clone https://github.com/junbin123/likeday.git

# 安装依赖包
npm i

# 本地开发，监听接口端口号 3000
npm run dev

# 本地开发，连接生产接口
npm run dev -prod

# 打包，连接生产接口
npm run build -prod
```
